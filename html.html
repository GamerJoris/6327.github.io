Wat is html?

Html is een afkorting van Hyper Text Markup Language. Dat betekent eigenlijk de taal van de websites. Html vertelt je browser hoe de website weergeven moet worden. Html is geen programmeertaal zoals vaak gedacht wordt. Je kan html goed verglijken met bouwstenen van een lego-set.
